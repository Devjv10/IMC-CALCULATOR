# IMC-CALCULATOR

Calculadora de IMC (Índice de Massa Corporal) simples feita com **HTML, CSS e JavaScript**.

---

## Funcionalidades

- Calcula o IMC a partir do peso e altura do usuário.
- Exibe a classificação do IMC (Abaixo do peso, Normal, Sobrepeso, Obesidade).
- Layout simples e responsivo.
- Feedback visual claro.

---

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Digite seu peso (kg) e altura (m).
3. Clique no botão de calcular.
4. Veja seu IMC e a classificação correspondente.
